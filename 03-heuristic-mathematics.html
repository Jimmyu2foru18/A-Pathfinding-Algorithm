Heuristic Mathematics

What is a Heuristic?

A heuristic is an educated guess about the cost to reach the goal from any given node.

Key Properties

  1. Admissible: h(n) вүӨ h*(n) (never overestimate)
  2. Consistent: h(n) вүӨ c(n,n') + h(n') (triangle inequality)
  3. Fast to compute: O(1) time complexity

Common Heuristics

Formula: h(n) = |xвӮҒ - xвӮӮ| + |yвӮҒ - yвӮӮ|

``` Example: From (1,1) to (4,3) h = |4-1| + |3-1| = 3 + 2 = 5

в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ в”Ӯ в”Ӯ в”ӮGв”Ӯ в”Ӯ G = Goal (4,3) в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”Ӯ в”Ӯ2в”Ӯ1в”Ӯ в”Ӯ Numbers = Manhattan distance в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”ӮSв”Ӯ3в”Ӯ2в”Ӯ1в”Ӯ S = Start (1,1) в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”Ӯ4в”Ӯ3в”Ӯ2в”Ӯ1в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ ```

Best for: 4-directional movement (no diagonal)

Nodes explored: Many (like Dijkstra) в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ*в”Ӯ*в”Ӯ*в”ӮGв”Ӯ в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ*в”Ӯ*в”Ӯ*в”Ӯ*в”Ӯ в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ*в”ӮSв”Ӯ*в”Ӯ*в”Ӯ в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ

Choosing the Right Heuristic

| Movement Type | Best Heuristic | Reason | |---------------|----------------|--------| | 4-directional | Manhattan | Matches actual movement | | 8-directional | Chebyshev | Accounts for diagonal | | Any direction | Euclidean | True geometric distance | | Weighted edges | Custom | Domain-specific |

Mathematical Proof: Admissibility

For A* to be optimal, the heuristic must be admissible:

Theorem: If h(n) вүӨ h(n) for all n, then A finds optimal solution.

Proof Sketch: 1. Suppose A returns suboptimal path with cost C 2. Optimal path has cost C < C 3. Some node n on optimal path must be in open set 4. f(n) = g(n) + h(n) вүӨ g(n) + h(n) = C 5. Since f(n) вүӨ C* < C, node n would be expanded before suboptimal goal 6. Contradiction! вҲҺ

Implementation Tips

```javascript // Manhattan distance for grid function manhattanDistance(a, b) { return Math.abs(a.x - b.x) + Math.abs(a.y - b.y); }

// Euclidean distance function euclideanDistance(a, b) { const dx = a.x - b.x; const dy = a.y - b.y; return Math.sqrt(dx * dx + dy * dy); }

// Chebyshev distance function chebyshevDistance(a, b) { return Math.max(Math.abs(a.x - b.x), Math.abs(a.y - b.y)); } ```
