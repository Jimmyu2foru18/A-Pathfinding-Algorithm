Heuristic Mathematics

What is a Heuristic?

A heuristic is an educated guess about the cost to reach the goal from any given node. It is the component that makes A* "informed."

Key Properties

  • Admissible: h(n) ≤ h*(n) (it never overestimates the true cost). This guarantees optimality.
  • Consistent: h(n) ≤ cost(n,n') + h(n'). This ensures the first path found to a node is the optimal one.
  • Efficiency: It must be fast to compute (ideally O(1)).

Common Heuristics

1. Manhattan Distance (L1 Norm)

Formula: h(n) = |x₁ - x₂| + |y₁ - y₂|

Used for grids allowing only 4-directional movement (up, down, left, right).

2. Euclidean Distance (L2 Norm)

Formula: h(n) = √[(x₁ - x₂)² + (y₁ - y₂)²]

Used for environments allowing any-direction movement (straight-line distance).

3. Chebyshev Distance

Formula: h(n) = max(|x₁ - x₂|, |y₁ - y₂|)

Used for environments allowing 8-directional movement (including diagonals) where diagonal movement costs the same as cardinal movement.
