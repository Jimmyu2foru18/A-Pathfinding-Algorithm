Code Walkthrough

Project Architecture

This repository provides a complete, interactive A* implementation. The core files are:

  • demo.html: The interactive interface for visualizing the algorithm.
  • visualization.js: The core logic managing the grid, the A* algorithm, and the animation state.
  • shared.css: A centralized design system ensuring consistent styling across all pages.

Core Visualizer Logic

The AStarVisualizer class in visualization.js orchestrates the entire process:

  • Grid State: Manages nodes, obstacles, and the open/closed sets.
  • Algorithm Loop: Implements the core A* logic: selecting the lowest f-cost node, exploring neighbors, and updating costs.
  • Visualization: A render loop (draw) updates the HTML5 canvas to show the current search progress in real-time.
