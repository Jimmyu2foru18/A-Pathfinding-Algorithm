Node Evaluation Process

The Core Formula: f(n) = g(n) + h(n)

This formula determines the order in which A* explores the graph. Every node evaluated is given an f value, which represents its total estimated cost.

  • g(n): The actual cost incurred from the start node to the current node n.
  • h(n): The estimated cost from node n to the goal node.

Step-by-Step Evaluation

  1. Initialize: Calculate the f value for the start node and add it to the openSet.
  2. Expand: While the openSet is not empty:
    • Pick the node with the lowest f value (current node).
    • If it is the goal node, path reconstruction begins.
    • Move the current node from the openSet to the closedSet.
  3. Neighbor Evaluation: For each neighbor of the current node:
    • If the neighbor is in the closedSet, ignore it.
    • Calculate a tentative g value.
    • If this new path to the neighbor is better (lower g), update the neighbor's g, h, f values, and set its parent to the current node.
