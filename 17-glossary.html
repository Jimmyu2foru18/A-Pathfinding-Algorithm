Glossary

A* Algorithm
A graph traversal and pathfinding algorithm that efficiently finds the shortest path by using a heuristic.
Admissible Heuristic
A heuristic that never overestimates the true cost to the goal, ensuring optimality.
Open Set (Frontier)
A priority queue of nodes discovered but not yet evaluated.
Closed Set (Explored)
A set of nodes that have already been evaluated.
Manhattan Distance
A distance metric for 4-directional grids: |x1 - x2| + |y1 - y2|.
