Search Algorithms Comparison

Uninformed vs Informed Search

Uninformed Search (Blind Search)

Algorithms that explore without knowledge of the goal location.

Breadth-First Search (BFS)

Exploration Pattern: в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ1в”Ӯ2в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ2в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ6в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ6в”Ӯ7в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ - Guarantees: Shortest path - Time: O(b^d) - Space: O(b^d) - Problem: Explores everywhere equally

Depth-First Search (DFS)

Exploration Pattern: в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ1в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ2в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ6в”Ӯ7в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ - Guarantees: None - Time: O(b^m) - Space: O(bm) - Problem: May find suboptimal paths

Informed Search (Heuristic Search)

Algorithms that use domain knowledge to guide exploration.

Greedy Best-First Search

Exploration Pattern (toward goal): в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”ӮGв”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”Ӯ в”Ӯ в”Ӯ4в”Ӯ3в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”ӮSв”Ӯ1в”Ӯ2в”Ӯ в”Ӯ в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ - Guarantees: None - Time: O(b^m) - Space: O(b^m) - Problem: Can get trapped by local optima

Dijkstra's Algorithm

Exploration Pattern (uniform cost): в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ4в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ6в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ3в”Ӯ2в”Ӯ3в”Ӯ4в”Ӯ5в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ2в”Ӯ1в”Ӯ2в”Ӯ3в”Ӯ4в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ1в”ӮSв”Ӯ1в”Ӯ2в”Ӯ3в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ - Guarantees: Optimal path - Time: O((V + E) log V) - Space: O(V) - Problem: Explores in all directions equally

A* Algorithm: The Best of Both Worlds

Formula: f(n) = g(n) + h(n)

  • g(n): Actual cost from start
  • h(n): Heuristic estimate to goal
  • f(n): Total estimated cost

A* Exploration Pattern: в”Ңв”Җв”¬в”Җв”¬в”Җв”¬в”Җв”¬в”Җв”җ в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”ӮGв”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”Ӯ в”Ӯ3в”Ӯ2в”Ӯ1в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”Ӯ в”Ӯ4в”Ӯ3в”Ӯ2в”Ӯ в”Ӯ в”ңв”Җв”јв”Җв”јв”Җв”јв”Җв”јв”Җв”Ө в”ӮSв”Ӯ1в”Ӯ2в”Ӯ в”Ӯ в”Ӯ в””в”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҙв”Җв”ҳ

Why A* is Optimal

  1. Admissible Heuristic: Never overestimates
  2. Consistent: Satisfies triangle inequality
  3. Complete: Always finds solution if exists
  4. Optimal: Finds shortest path
