Interactive Demo Guide

How to Use the Visualizer

The interactive demo allows you to experiment with A* in real-time.

  • Start/Goal: Right-click to set start/goal.
  • Obstacles: Left-click or drag to create walls.
  • Execution: Use the "Start Algorithm" button to begin.

Experimentation Scenarios

Try testing different heuristics to see how they change the exploration pattern (the orange/yellow cells). Notice how Manhattan, Euclidean, and Dijkstra produce vastly different search areas.
